Examples
========

Three runnable scripts ship in the ``examples/`` directory of the
source repository. Each is self-contained and demonstrates one piece
of the API.

.. note::

   These examples are vendor-neutral and target a generic GigE Vision
   camera. They focus on protocol-level operations (discovery,
   bootstrap, register reads, frame reception). Real camera workflows
   typically wrap them in a vendor-specific driver that knows the
   register layout for that camera's GenICam XML.

The example walkthroughs will be expanded here once the examples land
in :file:`examples/`.
